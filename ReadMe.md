# 基于OpenCV 并查集 链式前向星的建筑图识别系统 (二开) 作者:周景鑫 vx:zjxyyds0307

1. 用的技术: OpenCV 并查集 链式前向星
2. 用途: 给定点的坐标以及线的坐标后 可以识别出建筑内外壁 门边 房间等多种信息
3. 怎么跑起来: 百度配置OpenCV 把它配置好后 就可以正常运行了
4. 数据文件:  
   1. \\x64\\debug\\边.txt: 每行用于存储每一条边的起点与终点
   2. \\x64\\debug\\点.txt: 每行存储点的x,y坐标
   3. \\x64\\debug\\门点坐标.txt: 每行存储门的x,y坐标
